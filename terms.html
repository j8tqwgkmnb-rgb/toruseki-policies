利用規約

最終更新日: 2025年12月30日

1. 総則

本利用規約（以下「本規約」という）は、トルセキ（以下「当社」という）が提供する出席管理アプリケーション「トルセキ」（以下「本サービス」という）の利用条件を定めるものです。

1.1 規約の適用

本規約は、本サービスの利用に関する当社と利用者との間の権利義務関係を定めることを目的とし、利用者と当社との間の本サービスの利用に関わる一切の関係に適用されます。

1.2 規約への同意

利用者は、本サービスを利用することにより、本規約の全ての内容に同意したものとみなされます。本規約に同意いただけない場合は、本サービスをご利用いただけません。

1.3 個別規定との関係

本サービスに関し、本規約のほか、利用上の注意、ガイドライン等を定める場合があります。これらは本規約の一部を構成するものとします。本規約と個別規定の内容が矛盾する場合は、個別規定が優先するものとします。

2. サービスの内容

本サービスは、教育機関における出席管理を効率化することを目的としたモバイルアプリケーションです。

2.1 提供機能

  1. 生徒の出席・欠席・遅刻・早退の記録管理
  2. クラス、学年、学期の管理
  3. 時間割・授業スケジュールの作成・管理
  4. 出席状況の統計分析及びレポート機能
  5. データのクラウド同期及びバックアップ
  6. 複数端末での利用（同一アカウント）
  7. その他、当社が随時提供する機能

2.2 サービス提供の前提

本サービスの利用には、インターネット接続環境及び対応端末が必要です。これらの準備及び維持は利用者の責任において行うものとします。

2.3 オフライン機能

一部機能はオフラインでも利用可能ですが、データの同期にはインターネット接続が必要です。

3. 利用登録・アカウント

3.1 利用資格

  1. 本サービスは、教育機関に所属する教師、講師、教育関係者を対象としています
  2. 18歳以上であること（未成年者は保護者の同意が必要）
  3. 反社会的勢力に該当しないこと
  4. 過去に本規約違反等により当社からアカウント停止・削除の措置を受けていないこと

3.2 アカウント登録

  1. Google Sign-In又はSign in with Appleを使用して登録
  2. 正確かつ最新の情報を登録すること
  3. 虚偽の情報を登録した場合、アカウントを削除することがあります

3.3 アカウント管理の責任

  1. 利用者は、自己の責任においてアカウント情報を管理するものとします
  2. アカウント情報の第三者への貸与・譲渡・売買等は禁止します
  3. アカウント情報が第三者に使用されたことにより生じた損害について、当社は一切の責任を負いません
  4. アカウント情報の盗用、不正使用の疑いがある場合は、直ちに当社にご連絡ください

4. 禁止事項

利用者は、本サービスの利用にあたり、以下の行為を行ってはなりません。

  1. 法令又は公序良俗に違反する行為
  2. 犯罪行為に関連する行為
  3. 虚偽の情報を登録する行為
  4. 他の利用者、第三者又は当社の知的財産権を侵害する行為
  5. 他の利用者、第三者又は当社の名誉・信用を毀損し、又はプライバシーを侵害する行為
  6. 本サービスのネットワーク又はシステム等に過度な負荷をかける行為
  7. 本サービスの運営を妨害するおそれのある行為
  8. 不正アクセス行為、クラッキング行為
  9. 本サービスの脆弱性を探索する行為
  10. リバースエンジニアリング、逆コンパイル、逆アセンブル等の行為
  11. 他の利用者のアカウントを使用する行為
  12. 本サービスの利用権を第三者に譲渡、貸与、担保設定等する行為
  13. 営利目的で本サービスを利用する行為（教育機関での通常利用を除く）
  14. 反社会的勢力等への利益供与行為
  15. 宗教活動又は宗教団体への勧誘行為
  16. その他、当社が不適切と判断する行為

5. コンテンツ及びデータの取扱い

5.1 利用者コンテンツの定義

「利用者コンテンツ」とは、利用者が本サービスに入力、アップロードしたすべてのデータ（生徒情報、出席記録、授業情報等）を指します。

5.2 利用者コンテンツの所有権

利用者コンテンツの知的財産権は、利用者又は正当な権利者に帰属します。当社は、本サービスの提供に必要な範囲でのみ利用者コンテンツを利用します。

5.3 利用者の責任

  1. 利用者は、利用者コンテンツについて自らが投稿その他送信することについて適法な権利を有していることを保証するものとします
  2. 利用者コンテンツが第三者の権利を侵害していないことを保証するものとします
  3. 個人情報保護法その他関連法令を遵守し、生徒の個人情報を適切に管理する責任を負います

5.4 バックアップ

当社は、システム障害等に備えてバックアップを行いますが、データの完全性を保証するものではありません。利用者は、重要なデータについて自己の責任において別途バックアップを取得することを推奨します。

5.5 データの削除

当社は、以下の場合に利用者コンテンツを削除することができます。

  1. 利用者からの削除要請があった場合
  2. 本規約に違反すると判断した場合
  3. 本サービスの運営上必要と判断した場合
  4. アカウント削除後30日が経過した場合

6. 利用料金及び広告

6.1 無料サービス

本サービスは、現在及び将来においても、すべての機能を完全無料でご利用いただけます。利用料金、月額料金、年額料金等の課金は一切行いません。

6.2 広告の表示

本サービスは、将来的に広告を表示する可能性があります。広告の表示により、本サービスの無料提供を継続することができます。

6.2.1 広告の種類

表示される広告は、第三者の広告配信事業者により提供される場合があります。広告の内容は、当社が管理するものではありません。

6.2.2 広告の配信方法

広告は、本アプリ内の適切な箇所に表示されます。広告の表示頻度や位置は、当社が決定します。

6.2.3 広告に関する免責

当社は、広告の内容、広告主の商品・サービス、広告の正確性等について一切の責任を負いません。広告をクリックしたことにより生じた損害についても、当社は責任を負いません。

6.3 料金に関する変更

本サービスは、現在及び将来においても無料で提供されます。将来的に有料化する予定はありません。

7. サービスの変更・中断・終了

7.1 サービスの変更

当社は、利用者に事前に通知することなく、本サービスの内容を変更、追加又は廃止することができます。ただし、重要な変更については、可能な限り事前に通知するよう努めます。

7.2 サービスの中断

当社は、以下の場合、利用者に事前に通知することなく、本サービスの全部又は一部を一時的に中断することができます。

  1. 本サービスに係るシステムの保守、点検、修理等を行う場合
  2. 火災、停電、天災地変等の不可抗力により本サービスの提供が困難な場合
  3. システムに障害が発生した場合
  4. その他、当社が中断を必要と判断した場合

7.3 サービスの終了

当社は、3ヶ月前までに利用者に通知することにより、本サービスの提供を終了することができます。サービス終了時には、利用者データのエクスポート機能を提供するよう努めます。

7.4 責任の制限

本条に基づくサービスの変更、中断、終了により利用者に生じた損害について、当社は一切の責任を負いません。

8. 保証の否認及び免責

8.1 保証の否認

当社は、本サービスについて、以下の事項を保証しません。

  1. 本サービスが利用者の特定の目的に適合すること
  2. 本サービスが期待する機能・正確性・有用性を有すること
  3. 本サービスの利用が利用者に適用される法令に適合すること
  4. 本サービスが中断しないこと、エラーが発生しないこと
  5. 本サービスに欠陥が存在しないこと
  6. 本サービスが安全であること、第三者による不正アクセス等がなされないこと

8.2 免責事項

当社は、以下の事由により利用者に生じた損害について、一切の責任を負いません。

  1. データの消失、毀損、改ざん等
  2. 本サービスの利用又は利用不能により生じた損害
  3. 本サービスの中断、停止、終了、変更により生じた損害
  4. 利用者の端末環境、通信環境等に起因する損害
  5. 第三者による不正アクセス、ウイルス感染等による損害
  6. 利用者間又は利用者と第三者との間で生じた紛争
  7. 利用者が入力したデータに起因する損害

8.3 責任の上限

当社の責めに帰すべき事由により利用者に損害が生じた場合であっても、当社が利用者に対して負う損害賠償責任は、本サービスが無料で提供されていることを考慮し、合理的な範囲内に限定されます。

8.4 間接損害等の免責

当社は、いかなる場合も、特別損害、間接損害、付随的損害、派生的損害、逸失利益について責任を負いません。

9.3 禁止行為

利用者は、本サービス及びこれに含まれるコンテンツについて、以下の行為を行ってはなりません。

  1. 複製、譲渡、貸与、翻訳、翻案、改変、公衆送信等

  3. 商標の削除、改変
  4. その他、知的財産権を侵害する行為

10. 利用停止・契約解除

10.1 当社による利用停止

当社は、利用者が以下のいずれかに該当すると判断した場合、事前に通知することなく、本サービスの全部又は一部の利用を停止し、又はアカウントを削除することができます。

  1. 本規約のいずれかの条項に違反した場合
  2. 登録情報に虚偽の事実があることが判明した場合
  3. 本サービスの運営を妨害した場合
  4. 6ヶ月以上本サービスの利用がない場合
  5. 当社からの連絡に対して30日間応答がない場合
  6. 反社会的勢力等であるか、資金提供その他を通じて反社会的勢力等の維持、運営若しくは経営に協力若しくは関与する等反社会的勢力等との何らかの交流若しくは関与を行っていると当社が判断した場合
  7. その他、当社が本サービスの利用を適当でないと判断した場合

10.2 利用者による解約

利用者は、アプリ内の設定からアカウントを削除することにより、いつでも本サービスの利用を終了することができます。

10.3 解約等の効果

  1. 本サービスの利用が終了した場合、利用者は本サービスを利用する一切の権利を失います
  2. アカウント削除後30日が経過した場合、当社はすべてのデータを削除します
  3. データ削除後の復旧は一切できません

11.1 変更の権利

当社は、以下の場合に、利用者の承諾を得ることなく本規約を変更することができます。

  1. 本規約の変更が、利用者の一般の利益に適合するとき
  2. 本規約の変更が、契約をした目的に反せず、かつ、変更の必要性、変更後の内容の相当性等に照らして合理的なものであるとき

11.2 変更の通知

本規約を変更する場合、変更後の本規約の施行日及び変更内容を、本サービス内又は当社ウェブサイトでの掲示その他適切な方法により、施行日の30日前までに通知します。

11.3 変更の効力

変更後の本規約は、施行日から効力を生じるものとします。利用者が変更後の本規約の施行日以降に本サービスを利用した場合、変更後の本規約に同意したものとみなします。

本規約の準拠法は日本法とします。

本規約の終了後も、第5条（コンテンツ及びデータの取扱い）、第8条（保証の否認及び免責）、第9条（知的財産権）、第12条（一般条項）は有効に存続するものとします。
